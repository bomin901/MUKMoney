# 🍽️ MUKMoney (먹머니)
기분 기반 음식 추천과 음식 소비 기록을 관리할 수 있는 풀스택 웹 애플리케이션입니다.
사용자의 감정 상태에 따라 음식을 추천하고, 먹은 음식과 금액을 달력 형태로 기록할 수 있습니다.

---
## Project 소개
MUKMoney는 사용자의 기분 기반 음식 추천과 월별 음식 소비 관리를 돕는 웹 서비스입니다.
추천 받은 음식을 기록하고, 소비 금액을 달력 형태로 시각화하여 음식 소비 패턴을 확인할 수 있습니다.

---
## 주요 기능
### 1. 음식 추천
* 기분 이모지 선택 기반 음식 추천
* 랜덤 음식 추천
* 다시 추천 버튼
* 추천 음식 기록 저장

### 2. 먹달력
* 월별 달력 UI
* 날짜별 먹은 음식 표시
* 날짜별 소비 금액 표시
* 월별 총 음식 소비 금액 계산
* 가장 많이 먹은 음식 분석

### 3. 나의 기록
* 기분 선택
* 먹은 음식 기록
* 식당 이름 저장
* 금액 기록
* 음식 후기 작성
* 사진 업로드
* Google Maps 링크 저장

### 4. 사용자 설정 (온보딩)
* 못 먹는 음식 선택
* 알러지 음식 선택
* 좋아하는 음식 선택
* 매운맛 선호도 설정

## 기술 스택
### Frontend
* React (Vite)
* Tailwind CSS
* Axios
* date-fns

### Backend
* Python FastAPI
* SQLite
* SQLAlchemy
* Uvicorn

---
## 실행 방법
### 1. frontend 실행
```
cd frontend
npm install
npm run dev
```
### 2. backend 실행
```
cd backend
uvicorn main:app --reload
```
### 3. 브라우저 접속
```
http://localhost:5173
```
